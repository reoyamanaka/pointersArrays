## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Apple LLVM version 10.0.0 (clang-1000.10.44.4)
```sh
brew install g++
```
### Installation

1. Clone the repository
```sh
git clone https://github.com/reoyamanaka/pointersArrays.git
```
